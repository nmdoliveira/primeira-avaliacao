# Primeira Avaliação

Essa é minha resolução da [primeira avaliação](https://github.com/9Nama/primeira-avaliacao) da Nama.

## Run

```
ruby bin/nama_team
```

## Test

Primeiro, instale as dependências:

```
bundle install
```

ou

```
gem install rspec
gem install rubocop
```

Rode os testes com:

```
rspec
```

E verifique o estilo com:

```
rubocop
```
